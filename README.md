# Umbrella Customizer

A simple, lightweight umbrella customization tool built using **HTML, CSS, and JavaScript (no frameworks)**.
Users can upload a brand logo and preview it on umbrellas available in **three colors: Pink, Blue, and Yellow**.

## Features

* Upload a logo and instantly preview it on the umbrella.
* Switch umbrella colors using color swatches.
* Dynamic theme change based on selected color.
* Clean, responsive UI recreated as shown in the provided demo.
* Smooth logo placement at the correct bottom position.

## Folder Structure

```
/assets      → umbrella images, icons
style.css     → CSS file
script.js   → functionality
index.html   → Main interface
```

## How to Run

Just open **index.html** in the browser — no setup required.

## Notes

* Built strictly without any frontend libraries/frameworks as per requirement.
* Closely follows the demo’s design, behavior, and interactions.
